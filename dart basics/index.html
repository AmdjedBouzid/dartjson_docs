1. Introduction

Dart is a client-optimized language designed for building mobile, desktop, and web applications. It is the language used for Flutter development. This guide will take you through Dart's basic syntax, data types, null safety, operations, I/O operations, and object-oriented programming features.

2. Dart Code Structure

Dart code typically starts with a main() function, which is the entry point of every Dart application. Dart uses curly braces ({}) for code blocks.

void main() {
  print('Hello, Dart!');
}

The structure of a Dart program includes:

  • Importing libraries
  • Defining variables
  • Writing functions
  • Class definitions

3. Data Types

Dart supports a rich set of data types that can be categorized into:

  • Numbers (int, double)
  • Strings
  • Booleans
  • Lists (arrays)
  • Maps (key-value pairs)
  • Sets (unique collections of objects)

Example

int age = 30;
double height = 5.9;
String name = 'John';
bool isStudent = true;
List students = ['John', 'Alice', 'Bob'];
Map grades = {'Math': 90, 'Science': 85};

4. Null Safety

Null safety helps prevent errors caused by null values. In Dart, a variable cannot contain a null value unless explicitly declared as nullable using the ? operator.

String? name;  // Nullable string
name = 'John'; // Valid
name = null;    // Valid

To ensure safety, Dart forces checks to ensure that you do not access nullable variables unless they are properly checked for null.

5. Operations

Dart supports various operations, including arithmetic, relational, logical, and assignment operations:

Arithmetic Operations

int sum = 5 + 3;   // Addition
int diff = 5 - 3;  // Subtraction
int prod = 5 * 3;  // Multiplication
double div = 5 / 2; // Division

Relational Operations

bool isEqual = 5 == 3;  // Equality check
bool isGreater = 5 > 3;  // Greater than check

Logical Operations

bool and = true && false;  // Logical AND
bool or = true || false;    // Logical OR

6. I/O Operations

Dart provides libraries for performing input and output (I/O) operations, such as reading from and writing to the console or files.

Console I/O

import 'dart:io';

void main() {
  stdout.write('Enter your name: ');
  String? name = stdin.readLineSync();
  print('Hello, $name!');
}

The code above demonstrates how to read user input from the console and print a message to the screen.

File I/O

import 'dart:io';

void main() async {
  File file = File('example.txt');
  await file.writeAsString('Hello, Dart!');
  String contents = await file.readAsString();
  print(contents);  // Output: Hello, Dart!
}

7. Object-Oriented Programming (OOP)

Dart is an object-oriented programming language that supports classes, inheritance, interfaces, and more.

Classes and Objects

class Person {
  String name;
  int age;

  Person(this.name, this.age);

  void greet() {
    print('Hello, my name is $name and I am $age years old.');
  }
}

void main() {
  Person person = Person('John', 30);
  person.greet();
}

Constructors

class Animal {
  String name;

  Animal(this.name);  // Constructor

  void speak() {
    print('$name makes a sound.');
  }
}

Inheritance

class Dog extends Animal {
  Dog(String name) : super(name);

  @override
  void speak() {
    print('$name barks.');
  }
}

void main() {
  Dog dog = Dog('Rex');
  dog.speak();  // Output: Rex barks.
}

8. Conclusion

Dart is a powerful, easy-to-learn language that provides modern features like null safety and a rich set of data types. It supports object-oriented programming principles and is widely used for mobile development with Flutter. This documentation has covered Dart's basic syntax, operations, and object-oriented features. By understanding these concepts, you can build more robust applications using Dart.
